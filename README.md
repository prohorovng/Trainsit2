# Trainsit Wireframes — Vercel Ready

## What is included
- `index.html` — standalone static page with full-page wireframe previews
- `vercel.json` — minimal Vercel config

## Deploy to Vercel
1. Create a new GitHub repository.
2. Upload all files from this folder to the root of the repository.
3. Go to Vercel.
4. Import the GitHub repository.
5. Vercel will detect it as a static site automatically.
6. Click Deploy.

## Notes
- No build step is required.
- No package.json is required.
- The site runs as plain static HTML/CSS.
